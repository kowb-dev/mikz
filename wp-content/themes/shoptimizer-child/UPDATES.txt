ОБНОВЛЕНИЯ ПРОЕКТА SHOPTIMIZER CHILD v1.0.9
==============================================

РЕАЛИЗОВАННЫЕ ФУНКЦИИ:

1. КАСТОМНАЯ СИСТЕМА УВЕДОМЛЕНИЙ
   - При добавлении товара в корзину, избранное или сравнение появляется красивое уведомление
   - Уведомления отображаются в правом верхнем углу
   - Автоматическое закрытие через 5 секунд
   - Возможность закрыть вручную кнопкой X
   - Разные цвета для корзины (оранжевый), избранного (красный) и сравнения (синий)

2. ЗНАЧКИ СЧЕТЧИКОВ
   - Исправлена проблема: значок корзины теперь появляется СРАЗУ при первом добавлении товара
   - Добавлены значки к иконкам: Избранное и Сравнения в меню действий (header)
   - Значки показывают количество товаров с анимацией появления
   - Автоматическое обновление через AJAX при изменении количества
   - Разные цвета значков: Корзина (оранжевый), Избранное (красный), Сравнения (синий)

3. МОБИЛЬНАЯ ВЕРСИЯ /shop/
   - На странице магазина (/shop/) в мобильной версии теперь отображаются:
     * Цены товаров
     * Поле количества с кнопками +/-
     * Кнопка добавления в корзину с иконкой
   - Работает только на экранах менее 767px

4. СКРЫТИЕ УВЕДОМЛЕНИЯ YITH COMPARE
   - Принудительно скрыто уведомление "Добавлено" от плагина YITH Compare
   - Используется CSS !important и JavaScript для полного удаления элемента
   - Решение работает на всех страницах включая single product

5. ИСКЛЮЧЕНИЕ КАТЕГОРИИ MISC
   - Товары категории "misc" (ID: 253) теперь полностью исключены из:
     * Блока "Новинка в магазине" на странице корзины
     * Всех WooCommerce блоков и шорткодов
     * Результатов поиска
   - Применяется автоматически ко всем запросам товаров
